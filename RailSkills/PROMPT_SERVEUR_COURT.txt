Modifie le backend RailSkills pour ajouter ces endpoints. Le serveur est en production avec PM2.

## ENDPOINTS À AJOUTER

1. GET /api/health → retourne { status: 'ok', timestamp }

2. POST /api/sharepoint/token → obtient un token Azure AD pour SharePoint
   - Utilise axios pour appeler https://login.microsoftonline.com/{AZURE_TENANT_ID}/oauth2/v2.0/token
   - Envoie client_id, client_secret, scope, grant_type en x-www-form-urlencoded
   - Retourne { accessToken, expiresIn, tokenType }

3. GET /api/organization/secret → retourne le secret de chiffrement
   - Lit ORGANIZATION_SECRET et ORGANIZATION_NAME depuis .env
   - Retourne { secret, organizationName, updatedAt }

4. GET /api/organization → retourne { name: ORGANIZATION_NAME }

## VARIABLES .env À AJOUTER

```
AZURE_TENANT_ID=4a7c8238-5799-4b16-9fc6-9ad8fce5a7d9
AZURE_CLIENT_ID=bd394412-97bf-4513-a59f-e023b010dff7
AZURE_CLIENT_SECRET=<le_vrai_client_secret>
ORGANIZATION_NAME=SNCF Traction
ORGANIZATION_SECRET=<secret_de_chiffrement>
```

## DÉPENDANCE
npm install axios (si pas déjà installé)
