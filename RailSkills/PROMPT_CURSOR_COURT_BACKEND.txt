## ENDPOINTS À AJOUTER AU BACKEND

### 1. GET /api/health
Retourne { status: 'ok', timestamp, version }

### 2. POST /api/sharepoint/token
Obtient un token SharePoint via Azure AD.
- Utilise les variables d'environnement : AZURE_TENANT_ID, AZURE_CLIENT_ID, AZURE_CLIENT_SECRET
- Appelle https://login.microsoftonline.com/{tenantId}/oauth2/v2.0/token
- Retourne { accessToken, expiresIn, tokenType }

### 3. GET /api/organization/secret
Retourne le secret de chiffrement organisationnel.
- Lit le header X-CTT-Identity pour identifier le CTT
- Utilise les variables : ORGANIZATION_SECRET, ORGANIZATION_NAME
- Retourne { secret, organizationName, updatedAt }
